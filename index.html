Langkah-langkah upload via Google Drive:
  1. Buka drive.google.com → upload file gambar siteplan-mu
  2. Klik kanan file → "Share" / "Bagikan"
  3. Ubah ke "Anyone with the link" / "Siapa pun yg memiliki link" → permission "Viewer"
